# Examples

Explore various integration examples showcasing Motion Drawer's flexibility and
compatibility with different UI libraries. These examples demonstrate key features
like customizable snap points, offset, padding, border radius, and more.

<Callout type='info'>
  You must install `Motion` library `v12.23.24` or newer into your project as
  Motion Drawer uses Motion under the hood.
</Callout>
